# Bread Picking Game

Akashic Engineで制作したカジュアルゲームです。  
制限時間内に指定されたパンを選び、コンボを繋いでスコアを稼ぎます。

---

## 🎮 Gameplay

- 指定されたパンを選ぶとスコア加算
- 間違えるとコンボリセット
- コンボに応じてスコア増加
- 制限時間あり

---

## 🧠 Features

- ランダム生成（パンの出現）
- スコア・コンボ管理
- 制限時間管理
- シーン遷移（オープニング / ゲーム / リザルト）
- イベント駆動による入力処理

---

## 🛠 Tech Stack

- JavaScript
- Akashic Engine

---

## 💡 What I focused on

- コンボによるスコア設計でゲーム性を向上
- ランダム生成によりリプレイ性を確保
- シンプルな操作でも遊べる設計
